# Rocket League Stats API

API REST construida con FastAPI que consume la API de Ballchasing.com 
y devuelve estadísticas limpias y estructuradas de partidas de Rocket League.

## Endpoints

- GET / → Estado de la API
- GET /player/{player_name} → Últimas 10 partidas de un jugador

## Tecnologías

- FastAPI
- Pydantic v2
- Python 3.14
